Intro

Welcome to Bootstrap Mashup, an unholy Frankenstein's Monster site which exhibits a wide range of Bootstrap components, classes and examples, providing a not-so-delightful feast for your eyes!

Below, you'll find an entire plethora of components. If you'd like to see some example features, please click here.

The Alert Component

Hey, you! This is a customised Bootstrap alert component!

The "Alert" component is a fully responsive Bootstrap component which displays text in a resizable container. This container has been styled with a darkish red background, white text colour, no border radius, and a box shadow!

It's also important to note that any length of text can be squeezed into an alert component and it will stretch and respond accordingly!

The Badge Component

These are badge components with different styles!

The "Badge" component is an inline component which is typically used as a label or a counter! If a group of them are placed together like above, then they will responsively wrap and fit tidily on the page!

It's also important to note that badges scale to match the size of the immediate parent element, so if they're nested within a heading, they will grow to match the font size!

The Breadcrumb Component

  1. You
  2. Are
  3. Here!

The Breadcrumb component uses a "nav" element with an ordered list and list items which help users to navigate the page!

With the help of an "aria-current" attribute and an added "active" class, a breadcrumb item will be styled differently (like the "Here!" item), allowing users to easily know where they are on a page!
